# Data Structure Visualization (Stack & Queue) in C

An interactive graphical simulation that visualizes the fundamental operations of Stack and Queue data structures. This application was built using legacy C within the Turbo C++ IDE to make abstract data management visual and easy to understand.

## Features
- **Stack Operations Visualization:** Animates `Push` and `Pop` mechanics demonstrating Last-In, First-Out (LIFO) memory behavior.
- **Queue Operations Visualization:** Animates `Enqueue` and `Dequeue` operations demonstrating First-In, First-Out (FIFO) pipeline flows.
- **Dynamic Graphical Rendering:** Uses `graphics.h` rectangles, lines, and text fonts to simulate memory cells changing state in real-time.

## Technical Details
- **Language:** C (Legacy Standard)
- **Graphics Library:** Borland Graphics Interface (`graphics.h`)
- **Environment:** Turbo C++ 3.0 / DOSBox Emulator
